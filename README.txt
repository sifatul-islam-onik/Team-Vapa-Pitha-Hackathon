Cross-Platform Application with Android and Desktop Support
Overview
This project is a cross-platform application consisting of an Android application and a Desktop application. Both applications are designed to provide seamless functionality while utilizing different technologies tailored to their platforms.

Android Application: Developed in Java and uses Firebase as its backend database.
Desktop Application: Built using JavaFX, with data management through a MySQL database and JSON for data serialization.
Features
Android Application
Built with Java.
Real-time database operations using Firebase.
User-friendly and responsive UI optimized for mobile devices.
Secure authentication and data handling.
Desktop Application
Developed with JavaFX.
Uses MySQL for persistent data storage.
JSON support for exporting and importing data.
Rich desktop UI for enhanced user experience.
